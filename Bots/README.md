# HR-ассистент (Telegram бот)

Бот для управления вакансиями и кандидатами в Telegram.

---

## Возможности

- Добавление и просмотр вакансий
- Добавление и просмотр кандидатов
- Статистика по вакансиям и кандидатам

---

## Установка и запуск

### 1. Установите зависимости

bash

pip install aiogram python-dotenv

python HR-assistent.py
