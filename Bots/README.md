# ZaiP-dating bot (Telegram бот)

Бот для знакомств с системой анкет и мэтчинга. Пользователи создают анкеты, ищут пары, ставят лайки.

---

## Возможности

- Создание и редактирование анкеты (имя, возраст, город, описание, фото)
- Просмотр анкет других пользователей
- Система лайков и взаимных симпатий
- Чат после взаимного лайка
- Хранение данных в SQLite

---

## Стек технологий

- Python 3.9+
- python-telegram-bot
- SQLite
- python-dotenv
