# Кликеры — коллекция игр на Python

В этой папке собраны три версии игры «Кликер», реализованные на Python с разными механиками.

---

## 📁 Содержание папки

| Файл | Описание |
|------|----------|
| `Кликер v1.py` | Классический кликер: нажимай и набирай очки |
| `Расширенный кликер v2.py` | Кликер с улучшениями, магазином и пассивным доходом |
| `Кликер с имитацией человека v3.py` | Демонстрация автоматизации с эмуляцией пользователя |

---

## 1. Кликер v1 — классический

Самая простая версия. Каждое нажатие приносит одно очко.

### Как играть

- Запустите скрипт
- Нажимайте Enter (или любую клавишу) для клика
- Каждый клик = +1 очко
- Для выхода нажмите `q`

### Запуск

```bash
python "Кликер v1.py"
python "Расширенный кликер v2.py"
python "Кликер с имитацией человека v3.py"
