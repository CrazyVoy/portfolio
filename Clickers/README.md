# 🖱️ Бот-кликер (Auto Clicker)

Простой, но функциональный бот для автоматизации кликов мыши, написанный на Python. Позволяет выполнять клики с заданным интервалом, останавливаться по горячей клавише или экстренно — при перемещении мыши в угол экрана.

## 🎯 Возможности

- 🔁 **Автоматические клики** с настраиваемым интервалом (по умолчанию 0.1 сек)
- 🛑 **Остановка по горячей клавише** (настраивается, по умолчанию `F8`)
- ⚠️ **Экстренная остановка** — переместите мышь в верхний левый угол экрана
- ⌨️ **Прерывание по Ctrl+C** в консоли
- 📝 **Интерактивный запуск** — можно задать интервал и клавишу остановки при старте

## 📋 Требования

- Python 3.6 или выше
- Библиотеки:
  - `pyautogui` — управление мышью
  - `keyboard` — глобальный перехват клавиш

## ⚙️ Установка

1. **Клонируйте репозиторий** или скачайте файл `clicker.py` в удобную папку.

2. **Установите зависимости**:

```bash
pip install pyautogui keyboard
Примечание для Linux/macOS:

На Linux может потребоваться запуск от sudo для keyboard (или установка python3-xlib и python3-tk).

На macOS может потребоваться разрешить управление компьютером в настройках безопасности.

🚀 Использование
Интерактивный запуск (с параметрами)
bash
python clicker.py
Программа запросит интервал кликов (в секундах) и клавишу остановки. Можно просто нажать Enter, чтобы оставить значения по умолчанию.

Прямой запуск с параметрами в коде
Раскомментируйте строку внутри if __name__ == "__main__"::

python
simple_clicker(interval=0.05, stop_key='esc')
Затем выполните:

bash
python clicker.py
Программный вызов в своём коде
python
from clicker import simple_clicker

# Запустить кликер с интервалом 0.2 сек и остановкой по F2
simple_clicker(interval=0.2, stop_key='f2')
⌨️ Управление
Действие	Способ
Остановка	Нажать настроенную клавишу (по умолчанию F8)
Экстренная остановка	Переместить мышь в левый верхний угол экрана (0, 0)
Принудительная остановка	Нажать Ctrl+C в консоли
⚠️ Важные замечания
Экстренная остановка срабатывает только когда мышь достигает координат (0, 0). Если курсор не доходит до угла, остановка не произойдёт.

Глобальный перехват клавиш (keyboard) требует прав администратора/root на некоторых системах. На Linux может не работать без sudo.

Не мешайте работе бота — во время его работы старайтесь не двигать мышь и не кликать, иначе курсор может уйти в сторону.

Используйте ответственно. Боты-кликеры могут нарушать правила некоторых игр и сервисов.

🛠️ Возможные проблемы и решения
Проблема	Решение
ModuleNotFoundError: No module named 'pyautogui'	Установите библиотеку: pip install pyautogui
keyboard не работает на Linux	Запустите скрипт с sudo, либо установите python3-xlib и python3-tk
Бот не останавливается по клавише	Проверьте, что окно терминала активно (глобальный перехват работает в фоне, но может конфликтовать с некоторыми приложениями). Используйте экстренную остановку мышью или Ctrl+C
